# Marks Manager (Python Project)

This is a simple Python program that helps you store, view, and calculate the average of student marks.  
All records are saved in a text file named ⁠ marks.txt ⁠, so the data stays even after closing the program.

## Features
•⁠  ⁠Add a student's name and marks  
•⁠  ⁠View all saved records  
•⁠  ⁠Calculate average marks  
•⁠  ⁠Automatically loads marks from ⁠ marks.txt ⁠  
•⁠  ⁠Simple menu-based interface  

## How to Run
1.⁠ ⁠Make sure Python is installed.
2.⁠ ⁠Open the project folder in VS Code.
3.⁠ ⁠Run the program using:

```bash
python marks_manager.py
